# Newstrality
뉴스의 편향된 부분을 수정하거나 제거한 후 요악을 해주는 서비스입니다.
또한 뉴스에 대해 궁금한 점을 챗봇에 물어보고 정보를 얻을 수 있습니다.

## demo link
http://13.209.91.211:3000/

## usage
저희의 웹 콘텐츠를 보기 위해선 pc와 크롬의 다크 모드 설정을 끄셔야 합니다.    
이 문제에 대해 생각하지 못하여 현재 다크 모드로 할 경우에 텍스트가 보이지 않는 현상이 있습니다. 추후 수정 예정입니다.

# 사용 방법
1. 웹 페이지에서 정치/사회 면의 가장 최근 뉴스 기사를 10개를 확인 할 수 있다. 
2. 관심 있는 기사를 클릭하여 중립적으로 변환되어있는 기사 요약본을 확인 할 수 있다.
3. 원본 기사와 비교하여 중립을 유지하기 위하여 기사를 바꾼 이유에 대한 설명을 확인할 수 있다. 
4. 원본 기사와 변화된 기사 사이에 객관성에 대한 수치를 확인할 수 있다.

# 현재 구현된 사항
1. 사용자는 홈화면에서 해당 관심사 뉴스 중 최신 10개의 기사 헤드라인을 볼 수 있다. 
2. 중립적으로 변환된 기사에 대한 내용을 확인할 수 있다.


# 최종 목표 
대한민국 국민의 편향된 정치관 타파 및 건강한 정치관 확립

# 향후 계획
## 추가 기능 구현
1. 사용자는 외부 기사를 검색하고 해당 기사를 확인할 수 있다
2. 사용자는 로그인을 통해 회원 기능을 이용할 수 있다.
3. 사용자 로그인을 통해 사용자가 제공된 기사가 중립적이지 않다고 판단되는 경우 피드백을 남길 수 있다.
4. 사용자는 해당 언론사의 편향정도를 확인 할 수 있다.

## 성능 개선
계속하여서 gpt 프롬프트 엔지니어링을 통하여서 성능을 개선한다.
