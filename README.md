# Newstrality
뉴스의 편향된 부분을 수정하거나 제거한 후 요악을 해주는 서비스입니다.
또한 뉴스에 대해 궁금한 점을 챗봇에 물어보고 정보를 얻을 수 있습니다.

## demo link
http://13.209.91.211:3000/

## usage
저희의 웹 콘텐츠를 보기 위해선 pc와 크롬의 **다크 모드 설정을 끄셔야** 합니다.    
이 문제에 대해 생각하지 못하여 현재 다크 모드로 할 경우에 텍스트가 보이지 않는 현상이 있습니다. 추후 수정 예정입니다.   
감사합니다.

# 사용 방법
1. 웹 페이지에서 정치/사회 면의 가장 최근 뉴스 기사를 10개를 확인 할 수 있다. 
2. 관심 있는 기사를 클릭하여 객관적으로 변환되어있는 기사 요약본을 확인 할 수 있다.
3. 원본 기사와 비교하여 객관성을 유지하기 위하여 기사를 바꾼 이유에 대한 설명을 확인할 수 있다. 
4. 원본 기사와 변화된 기사 사이에 객관성에 대한 수치를 확인할 수 있다.


   사용자가 부적절한 질문 혹은 기사를 입력할 경우를 방지하도록 하였습니다. 모더레이션을 이용하여서 기사 관련 질문을 묻는 항목과 사용자가 기사를 입력하는 부분에 적용하였습니다
   댓글 기능의 경우 유일한 사용자의 피드백을 받을 수 있는 수단이기 때문에 모더레이션 수치를 낮춰서 진행하였습니다. 사용자의 피드백을 받아서 최대한 자유롭게 토론할 수 있는 공간을 만들고 추구하고자 하였습니다.
